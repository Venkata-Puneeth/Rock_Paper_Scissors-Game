# ✊ Rock Paper Scissors Game

A user-friendly Rock–Paper–Scissors game built using Python and CustomTkinter.  
This application provides an interactive graphical interface where users can play against the computer with real-time feedback and score tracking.

---

## 🚀 Features

- 🎮 Interactive GUI with buttons for Rock, Paper, and Scissors  
- 🤖 Random computer choice generation  
- 🧠 Game logic implementation:
  - Rock beats Scissors  
  - Scissors beat Paper  
  - Paper beats Rock  
- 🏆 Result display (Win / Lose / Tie)  
- 📊 Score tracking for multiple rounds  
- 🔄 Reset game functionality  
- 🎨 Clean and modern user interface  

---

## 🛠️ Tech Stack

- Python  
- CustomTkinter  
- Random module
